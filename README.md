# HiDnsLog

```
一个可查询和管理DNS解析记录的工具，支持DNSlog 私有化部署，即使用自己的域名。

支持完整的DNS Server 功能，支持自动从其它DNS Server返回解析记录。

支持Burp Collaborator，支持SSRF跳转漏洞测试。

支持域名解析自动二次绑定（绕过一些检测）。


```
